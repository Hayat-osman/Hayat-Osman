### Featured Projects

#### 🚀 [Production-Grade Gatus Monitoring Platform - ECS Fargate](https://github.com/Hayat-osman/gatus-ecs-project)

Highly available monitoring platform deployed on AWS ECS Fargate. Built to production standards, not a tutorial.

- 9 reusable Terraform modules covering VPC, ECS, ALB, IAM, ACM, DNS, CloudWatch, autoscaling and security groups
- Chainguard distroless images and multi-stage Docker builds cutting image size by 93% with zero CVEs
- 6 GitHub Actions workflows with OIDC and risk-tiered IAM roles gated by branch protection
- Grype, Checkov and TFLint integrated into CI blocking HIGH and CRITICAL vulnerabilities before ECR push
- S3 Gateway Endpoint to keep ECR image pulls inside the VPC
- ALB health checks with HTTP 200 validation as foundational SLI signals for service availability

`ECS Fargate` `Terraform` `Docker` `GitHub Actions` `OIDC` `CloudWatch` `ECR` `ALB` `ACM` `Route 53`
